# Java Lab Projects

This repository contains several Java lab projects created for coursework. The files demonstrate basic Java programming concepts, user interaction, and simple GUI applications using Swing.

## Project Files

- `HelloWorld.java`
  - A simple program that prints `Hello World!` to the console.

- `lab2.java`
  - Reads the user's name from the console and prints a personalized greeting.
  - Demonstrates basic input handling with `Scanner`.

- `calculator.java`
  - A Swing-based graphical calculator for adding, subtracting, multiplying, and dividing two numbers.
  - Includes a basic GUI layout with input fields, buttons, and a result display.

- `NewspaperCoverPage.java`
  - A Swing GUI application that displays a mock newspaper cover page.
  - Demonstrates layout management, fonts, panels, and styled text.

## How to Compile

Open a terminal in the project folder and run the Java compiler for each file:

```bash
javac HelloWorld.java
javac lab2.java
javac calculator.java
javac NewspaperCoverPage.java
```

## How to Run

Run each compiled class using the Java command:

```bash
java HelloWorld
java lab2
java calculator
java NewspaperCoverPage
```

## Notes

- `HelloWorld` is a console example for basic Java output.
- `lab2` uses `Scanner` to accept user input and print a greeting.
- `calculator` is a simple GUI application with buttons for arithmetic operations.
- `NewspaperCoverPage` is a GUI design exercise with a newspaper-style layout.

## Assignment Summary

This lab project set is designed to practice basic Java programming and GUI creation. It covers:

- Console output and input
- Simple control flow and user interaction
- Java Swing components and layout managers
- Event handling for GUI buttons
